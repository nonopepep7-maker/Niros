# Niros
NirOS 1.0 — Python Desktop Environment
Welcome to NirOS, a lightweight, aesthetic desktop simulation built entirely in Python using the tkinter library. It’s designed to provide a "mini-OS" experience within a single window, complete with a login system, taskbar, and several built-in applications.

🚀 Features
Secure Authentication: A custom login screen to keep your "session" private.

Dynamic Desktop: A functional taskbar with a real-time clock and a toggleable Start Menu.

Window Management: Moveable and closable app windows with a smooth, interactive feel.

Dual Themes: Support for both Dark Mode (default) and Light Mode.

Built-in Apps:

Terminal: Execute basic commands like whoami, time, and echo.

Notepad: A simple text editor for quick notes.

Calculator: Handles basic arithmetic operations.

Settings: Change the system name and toggle themes.

System Info: View "hardware" specs and track your current session uptime.
